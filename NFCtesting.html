Info

The result of the NFC read will call the readNFCResult(data, id, tag) function.

You must define this function in your web app.

Example definition:

function readNFCResult(data, id, tag) {
  alert(`Data: ${data}\nID: ${id}\nTag: ${tag}`);
}
